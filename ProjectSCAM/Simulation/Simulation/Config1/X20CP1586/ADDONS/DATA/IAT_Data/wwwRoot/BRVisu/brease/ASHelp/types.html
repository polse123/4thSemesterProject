brease.config.MeasurementSystemFormat Definition of the format conversion depending on the MeasurementSystem (metric, imperial, imperial-us).
The format is an object with key value pairs for each MeasurementSystem.
The value is an object with the format description
The format description has the keys:
- decimalPlaces: number of decimal places
- minimumIntegerDigits: minimum number leading integer digits
- maximumIntegerDigits (optional): maximum number of leading integer digits. For higher numbers display mode switches to normalized scientific notation.
Example:
{'metric':{ 'decimalPlaces': 0, 'minimumIntegerDigits':1 }, 'imperial':{ 'decimalPlaces': 0 }, 'imperial-us':{ 'decimalPlaces': 2 }}
brease.config.MeasurementSystemUnit Definition of the unit conversion depending on the MeasurementSystem (metric, imperial, imperial-us).
The format is an object with key value pairs for each MeasurementSystem. The value is the common code for the unit.
Example:
{'metric':'CEL','imperial':'FAH','imperial-us':'KEL'}

ZoomType Zoom mode for scaling an image.
Possible values:
- original: original size of the image
- contain: Scale the image to the largest size such that both its width and its height can fit inside the content area
- cover: Scale the background image to be as large as possible so that the background area is completely covered by the background image. Some parts of the background image may not be in view within the background positioning area

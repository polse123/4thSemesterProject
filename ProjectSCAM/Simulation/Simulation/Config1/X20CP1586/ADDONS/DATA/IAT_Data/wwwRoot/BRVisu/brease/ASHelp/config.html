    brease.config.MeasurementSystemFormat

    brease.config.NumberFormat for every measurement system.

    Example:

      {'metric':{ 'decimalPlaces': 2, 'minimumIntegerDigits':2 }, 
  'imperial':{ 'decimalPlaces': 3, 'minimumIntegerDigits':1 }, 
  'imperial-us':{ 'decimalPlaces': 1, 'minimumIntegerDigits':1 }}
    • imperial: brease.config.NumberFormat
    • imperial-us: brease.config.NumberFormat
    • metric: brease.config.NumberFormat
    brease.config.MeasurementSystemUnit

    brease.config.UnitCode for every measurement system.

    Example:

      {'metric': 'GRM','imperial': 'ONZ','imperial-us': 'ONZ'}

    Unit codes are CEFACT Common Codes.
    • imperial: brease.config.UnitCode
    • imperial-us: brease.config.UnitCode
    • metric: brease.config.UnitCode
    brease.config.NumberFormat

    NumberFormat object in string notation: e.g.

      { 'decimalPlaces': 2, 'minimumIntegerDigits':2 }
    • decimalPlaces: Integer

      The number of fraction digits to use.
      0 <= decimalPlaces <= 20

    • maximumIntegerDigits: Integer

      The maximum number of integer digits to use. For higher numbers display mode switches to normalized scientific notation.
      maximumIntegerDigits >= 1.

    • minimumIntegerDigits: Integer

      The minimum number of integer digits to use.
      minimumIntegerDigits >= 1.
    brease.config.SortFunctions

    brease.config.UnitCode

  WTT         (for W)
